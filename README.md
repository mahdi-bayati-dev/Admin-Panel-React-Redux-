# 🛠️ Admin Panel - React + Redux + Node.js

پنل ادمین حرفه‌ای برای مدیریت محصولات، کاربران، مقالات و ... ساخته‌شده با React در فرانت‌اند و Node.js در بک‌اند.

---

## ✨ ویژگی‌ها

- 📦 افزودن، ویرایش و حذف محصولات
- 📝 مدیریت مقالات با قابلیت ویرایش و حذف
- 👥 مدیریت کاربران (مشاهده، حذف، ویرایش نقش)
- 📁 ساختار تمیز و ماژولار
- ⚙️ مدیریت وضعیت با Redux Toolkit
- 🌐 طراحی کاملاً ریسپانسیو

---

## 🧰 تکنولوژی‌ها

### فرانت‌اند:
- React
- Redux Toolkit
- React Router



### بک‌اند:
- Node.js
- Express.js

---

## 🚀 شروع پروژه

### نصب فرانت‌اند:

```bash
npm install
npm run dev
```

### نصب بک‌اند:

```bash
cd server
npm install
npm run dev
```
